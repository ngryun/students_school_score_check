# 고등학교 내신 분석 프로그램 Lite

NEIS(교육행정정보시스템) XLS 데이터를 분석하여 학생 학업 성취도를 시각화하는 웹 애플리케이션입니다.

## 주요 특징

- **완전한 클라이언트 사이드 처리**: 모든 데이터 처리가 브라우저에서 이루어져 개인정보가 외부로 유출되지 않습니다
- **다중 파일 지원**: 여러 학급의 성적 데이터를 동시에 분석할 수 있습니다
- **시각화**: Chart.js를 활용한 직관적인 성적 분포 차트 및 분석 그래프
- **등급 변환**: 5등급제와 9등급제 간 자동 변환 기능
- **다양한 내보내기 옵션**: PDF, CSV 형식으로 분석 결과 저장

## 기능

### 1. 파일 업로드
- 드래그 앤 드롭 또는 클릭하여 NEIS XLS 데이터 파일 업로드
- 다중 파일 동시 업로드 지원

### 2. 성적 분석
- 과목별 평균 및 성적 분포 분석
- 학생별 성취도 분석
- 산점도 및 막대 그래프를 통한 시각화
- 5등급제 ↔ 9등급제 자동 변환

### 3. 결과 내보내기
- **개별 학생 PDF 리포트**: 학생별 성적 분석 결과
- **전체 학급 PDF 일괄 생성**: 모든 학생의 리포트를 한 번에 생성
- **CSV 데이터 내보내기**: 외부 분석 도구에서 활용 가능한 형식

## 사용 방법

### 1. 실행
웹 브라우저에서 `index.html` 파일을 열거나, 정적 웹 서버를 통해 실행하세요.

```bash
# 예시: Python을 사용한 로컬 서버 실행
python -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속

### 2. 데이터 준비
NEIS에서 성적 데이터를 다운로드하세요:

1. NEIS 로그인
2. **성적조회/통계** → **학기말성적조회** → **학기말성적종합일람표**
3. **XLS data** 형식으로 다운로드 (주의: XLS가 아닌 XLS data 형식)

### 3. 분석 실행
1. 다운로드한 XLS data 파일을 업로드 영역에 드래그 또는 클릭하여 업로드
2. "분석 시작" 버튼 클릭
3. 분석 결과 확인

### 4. 결과 활용
- **취합용 DB 파일 생성**: 전체 데이터를 CSV 형식으로 내보내기
- **분석결과 HTML 저장**: 현재 분석 결과를 HTML 파일로 저장
- 개별 학생 리포트를 PDF로 생성

## 기술 스택

### 프론트엔드
- HTML5, CSS3, Vanilla JavaScript
- 반응형 디자인

### 라이브러리 (CDN)
- **[SheetJS (xlsx)](https://sheetjs.com/)**: Excel 파일 파싱
- **[Chart.js](https://www.chartjs.org/)**: 데이터 시각화
- **[JSZip](https://stuk.github.io/jszip/)**: 파일 압축
- **[jsPDF](https://github.com/parallax/jsPDF)**: PDF 생성
- **[html2canvas](https://html2canvas.hertzen.com/)**: HTML-to-Canvas 변환

## 파일 구조

```
student_school_record_analytics/
├── index.html          # 메인 UI 및 구조
├── script.js           # 핵심 애플리케이션 로직 (ScoreAnalyzer 클래스)
├── style.css           # 스타일링 및 반응형 디자인
├── CLAUDE.md           # 개발자를 위한 프로젝트 가이드
└── README.md           # 이 파일
```

### 핵심 클래스: ScoreAnalyzer (script.js)

주요 메서드:
- `analyzeFiles()`: 업로드된 Excel 파일 처리
- `displayResults()`: 분석 결과 렌더링
- `exportToCSV()`: 통합 데이터 CSV 내보내기
- `convertTo9Grade()`: 5등급제를 9등급제로 변환

## 등급 시스템

한국 교육 시스템의 등급 체계를 지원합니다:

- **5등급제** (표준): 1등급(최우수) ~ 5등급(최하)
- **9등급제**: 대학 입시용 변환 등급

## 보안 및 프라이버시

이 애플리케이션은 최대한의 프라이버시를 보장하도록 설계되었습니다:

- 모든 데이터 처리는 **클라이언트 측(브라우저)**에서만 이루어집니다
- 업로드된 파일은 **서버로 전송되지 않습니다**
- 외부 서버와의 통신이 **전혀 없습니다** (CDN 라이브러리 로드 제외)
- 페이지를 닫거나 새로고침하면 메모리의 모든 데이터가 삭제됩니다

## 브라우저 호환성

최신 버전의 Chrome, Firefox, Safari, Edge에서 정상 작동합니다.

## 개발

빌드 과정이 필요 없는 정적 웹 애플리케이션입니다. 개발을 위해서는:

1. 저장소 클론
2. `index.html` 파일을 브라우저에서 열기
3. NEIS에서 다운로드한 샘플 XLS data 파일로 테스트
